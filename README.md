## Local Webhook Development (Clerk)

To test Clerk webhooks locally, you must expose your local development server to the internet using `ngrok`.

Because Vinxi/TanStack Start enforces strict host-header checking, you must explicitly instruct ngrok to rewrite the host header and bypass the default browser warning page.

### 1. Start the Tunnel

Run the following command in your terminal while your local dev server (`pnpm run dev`) is running on port 3000:

```bash
ngrok http 3000 --host-header="localhost:3000" --request-header-add "ngrok-skip-browser-warning: true"
```

### 2. Update Clerk Dashboard

1. Copy the forwarding URL generated by ngrok (e.g., `https://ngrok-free.app`).
2. Go to your **Clerk Dashboard > Webhooks > Add Endpoint**.
3. Paste the URL and append the webhook path: `https://ngrok-free.app/api/webhooks/clerk`.
